# ☕ Coffee Sales Analysis (Interactive Dashboard using MS Excel)

## 📌 Project Objective

The Coffee Shop aims to analyze its sales performance using an interactive dashboard. This project helps the owner understand customer purchasing behavior, identify top-performing products, analyze sales trends across different time periods, and compare store performance to make better business decisions and improve overall sales.

## 📂 Dataset
The dataset used in this project includes coffee shop transaction data with attributes such as product details, sales amount, date & time, and store location. It is well-suited for analyzing sales trends, customer behavior, and business performance.

🔗 Dataset Link: https://drive.google.com/file/d/1oZGTtLaAEpI2Gy0gWxUk1YYAsi-DYcnr/view


## 📊 Key Performance Indicators (KPIs)

* **Total Sales Revenue** – Overall revenue generated from coffee sales

* **Total Quantity Sold** – Total number of products sold

* **Total Transactions** – Total number of customer purchases

* **Top 10 Products by Sales** – Identifies the best-selling products based on quantity

* **Sales by Product Category** – Analyzes which category (Coffee, Tea, Bakery, etc.) contributes most to sales

* **Sales by Product Type** – Breaks down performance by specific product types

* **Hourly Sales Trend** – Tracks sales performance across different hours of the day to identify peak business hours

* **Sales by Day & Month** – Analyzes trends using time-based filters

* **Location-wise Sales Performance** – Compares sales across different store locations

* **Size-wise Sales Distribution** – Shows customer preference for product sizes (Small, Regular, Large)

## ⚙️ Process

1. **Data Collection**

   * Imported the coffee sales dataset into Microsoft Excel.

2. **Data Cleaning**

   * Checked for missing values and removed blanks
   * Formatted date and time columns
   * Ensured data consistency for accurate analysis

3. **Data Transformation**

   * Created new columns such as Month Name, Day Name, and Hour
   * Organized data for better analysis using structured format

4. **Data Analysis (Pivot Tables)**

   * Built multiple pivot tables to analyze:

     * Product-wise sales
     * Category-wise distribution
     * Location-wise performance
     * Hourly sales trends

5. **Dashboard Creation**

   * Designed an interactive dashboard using charts and visuals:

     * Column charts
     * Bar charts
     * Line charts with markers
     * Pie charts

6. **Adding Interactivity**

   * Used slicers for Month and Day filters
   * Enabled dynamic data exploration

7. **Insights Generation**

   * Identified top-performing products
   * Analyzed peak sales hours
   * Compared store performance
   * Evaluated customer preferences
